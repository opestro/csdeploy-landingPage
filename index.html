Common Questions

What can I deploy?

CSDeploy handles PHP applications, Node.js projects, and static files with equal elegance. Built for versatility, engineered for performance.

How does file tracking work?

Advanced hash-based file tracking ensures only modified files are deployed. Customize your deployment scope with powerful ignore patterns.

Can I automate deployment?

Absolutely. Define your deployment workflow using intuitive Docker-like syntax in .deploycommands. From simple file moves to complex build processes.

Begin Your Journey

1. Installation

npm install -g csdeploy

2. Initialize Project

csdeploy init

Creates .cscc-ignore and .deploycommands files

3. Deploy

csdeploy deploy
