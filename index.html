Frequently Asked Questions

What protocols are supported?

CSCC Auto-Deploy supports FTP, SFTP, and SSH protocols with automatic detection.

How does the rollback feature work?

CSCC Auto-Deploy maintains a version history of your deployments. If issues arise, you can instantly revert to any previous version using a single command: 'csdeploy rollback'.

Can I use it with my CI/CD pipeline?

Yes! CSCC Auto-Deploy integrates seamlessly with popular CI/CD platforms like GitHub Actions, GitLab CI, and Jenkins through our CLI interface and configuration files.

What about environment variables and secrets?

We provide secure handling of environment variables and secrets through encrypted storage and integration with popular secret management services.
